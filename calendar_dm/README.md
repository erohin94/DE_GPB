В рамках сбора витрины данных необходимо было расчитать атрибуты.

Есть источники с взаимодейсвиями пользователя user_id и объекта object_id. 

Взаимодейсвия могут по встречам, лайкам, звонкам, департаменту.

Витрина обновляется каждую неделю, инкрементально, без удаления старых даных.
Расчет необходимо делать за 4 недели. Тоесть если обновляем витрину в понедельник, то берем четыре прошедших недели.
В витрине отчетные неделя это конец недели, тоесть восресенье.

Например обновляем витрину в 03.11.2025, значит ищем взаимодейсвия за четыре отчетных недели с 12.10.25, 19.10.25, 26.10.25, 02.11.25.
В отчетную неделю входят все взаимоджейсвия с понедельника по воскресенье
